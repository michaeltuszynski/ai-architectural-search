## ğŸ”§ Troubleshooting

### No Results Found
**Possible Causes:**
- Query too specific or unusual
- Architectural feature not in dataset
- Spelling errors in query

**Solutions:**
- Try broader terms (e.g., "glass" instead of "tempered glass curtain wall")
- Check spelling and try synonyms
- Use simpler descriptions

### Poor Quality Results
**Possible Causes:**
- Query too vague or ambiguous
- Conflicting features in query
- Non-visual descriptors used

**Solutions:**
- Be more specific about visual features
- Focus on one main characteristic
- Use architectural terminology

### Slow Loading
**Possible Causes:**
- High server load
- Large image processing
- Network connectivity

**Solutions:**
- Wait a moment and try again
- Check your internet connection
- Try during off-peak hours
